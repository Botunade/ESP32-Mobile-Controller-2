# Tank Controller Project

This project merges the ESP32 firmware and the React Native mobile app into a single repository.

## Structure

- **firmware/**: ESP32 firmware code (PlatformIO project).
- **mobile-app/**: React Native mobile application (Expo).
- **hardware-test/**: Simple hardware test sketches.
- **firmware-future/**: Experimental/future firmware features.

## Getting Started

### Firmware
Navigate to `firmware/` and use PlatformIO to build and upload.

### Mobile App
Navigate to `mobile-app/`, install dependencies with `npm install`, and run `npx expo start`.
